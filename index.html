step1

html/css

웹 페이지 구축의 기본! 깔끔한 레이아웃과 반응형 디자인으로 실전에서도 통하는 UI를 구현하고, CSS 애니메이션을 통해 감각적인 사용자 경험까지 완성해요. 여기에 더해, AI가 마크업에 미치는 영향과 허점 분석까지 배우며, 최신 AI 툴을 능숙하게 다루는 실전 감각을 키워요.

HTML 기초 및 웹 접근성

  • • 웹 표준과 웹 접근성, 시맨틱 마크업
  • • 전역 속성, 문서 메타데이터 요소
  • • 섹션/그룹핑/텍스트 레벨 요소
  • • 폼 요소, 테이블 요소, 스크립팅 요소
  • • WAI-ARIA: 역할(Role), 속성(Properties), 상태(States)

CSS 스타일링 및 레이아웃

  • • CSS 기초: 선택자, 상속, 우선순위, 타이포그래피
  • • 박스 모델, 리스트/배경 스타일링
  • • 모던 레이아웃: Flexbox, Grid
  • • 반응형 웹 디자인: 유연한 그리드, 이미지, 미디어 쿼리, 컨테이너 쿼리

Tailwind CSS & UI 프레임워크

  • • Tailwind CSS 설치 및 개발 환경 구성
  • • 유틸리티 클래스 스타일 및 활용
  • • 테마 커스터마이징, 커스텀 폰트, 컨테이너
  • • UI 프레임워크 컴포넌트 UI 제작 실습

step2

javascript

정적인 웹사이트는 이제 그만! JavaScript로 동적인 기능을 추가해 더 생동감 있는 웹을 만들어요. ES6+ 문법과 TypeScript까지 익히면 더 안전하고 유지보수하기 쉬운 코드 작성도 문제없어요.

JavaScript 기초

  • • 자바스크립트란? 개발 환경 구성
  • • 함수:정의, 호출, 매개변수, 반환값, 클로저, IIFE
  • • DOM 접근/조작, 이벤트 모델, 이벤트 위임
  • • AJAX, Fetch API, Axios 활용

JavaScript 심화

  • • 객체 지향: 프로토타입, 클래스, 상속(확장)
  • • 성능 최적화: 커링, 고차 함수, 메모이제이션
  • • 테스트: Jest, Vitest
  • • TypeScript 개요, 타입, 인터페이스

step3

react

프론트엔드의 핵심 기술 React로 컴포넌트 기반 개발을 익히고, CSR과 상태 관리까지 학습해 실무에 바로 적용 가능한 역량을 키워요. 여기에 AI를 활용한 코드 리뷰, 리팩토링, 개발 실습을 더해 AI와 함께 효율적이고 품질 높은 코드를 작성하는 실전 개발 능력을 완성해요.

React 펀더멘탈

  • • React란? 개발환경 설정 (Bun & Vite 등)
  • • JSX, 가상 DOM, 렌더링 프로세스 이해
  • • 리액트 훅: useState, useEffect, useRef 등
  • • 성능 최적화: useMemo, useCallback, memo

React 스타일링 &데이터 페칭

  • • 스타일 방법론: CSS, CSS Modules, Tailwind CSS 등
  • • 비동기 리액트: Promise / Suspense & useTransition
  • • 리액트 액션: useActionState, useOptimistic
  • • 에러 핸들링: ErrorBoundary / 예외 처리 컴포넌트

Next.js 프레임워크

  • • 렌더링 바운더리 (서버/클라이언트 컴포넌트)
  • • 라우팅 기본 & 고급 (인터셉팅, 패럴럴 라우트)
  • • Next.js 캐시 (4가지 타입)
  • • 정적/동적 렌더링, 증분 함수

EST 캠프 수강생만을 위한 혜택

  • 개강 전 온라인 학습 컨텐츠 제공

  • 고사양 노트북 무료 지원

  • 현직자 커리어 코칭 취업특강
